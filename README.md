# 🌾 Campo Direto

### Conectando a produção rural ao mercado digital

O **Campo Direto** é uma plataforma desenvolvida para facilitar a comercialização de produtos de pequenos produtores rurais, eliminando intermediários desnecessários e conectando-os diretamente a compradores finais. O projeto une tecnologia à produtividade rural em Minas Gerais.

---

## 🎯 O Problema
Pequenos produtores enfrentam dificuldades para escoar sua produção e obter preços justos devido à falta de canais de venda direta e à dependência de atravessadores.

## 💡 A Solução
Uma aplicação web robusta que permite o cadastro de produtos, gestão de estoque e visualização para compradores, trazendo transparência e agilidade ao agronegócio familiar.

---

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python
- **Framework Web:** Flask
- **Banco de Dados:** SQLite (escolhido pela sua leveza e facilidade de implementação inicial)
- **Frontend:** HTML5, CSS3 e Bootstrap para responsividade.

---

## 🚀 Funcionalidades
- [ ] Cadastro e login de produtores.
- [ ] Listagem de produtos disponíveis para venda.
- [ ] Painel administrativo para gestão de ofertas.
- [ ] Integração com conceitos de Software Engineering e boas práticas de código.

---

## ⚙️ Como Rodar o Projeto
1. Clone o repositório: `git clone https://github.com/seu-usuario/campo-direto.git`
2. Crie um ambiente virtual: `python -m venv venv`
3. Ative o venv:
   - Linux: `source venv/bin/activate`
   - Windows: `venv\Scripts\activate`
4. Instale as dependências: `pip install -r requirements.get`
5. Execute a aplicação: `flask run`
