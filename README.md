# 🚜 Campo Direto

O **Campo Direto** é um projeto de uma plataforma marketplace desenvolvida para conectar pequenos produtores rurais diretamente ao consumidor final, eliminando intermediários e valorizando o comércio local.

## 🚀 Funcionalidades
- **Listagem de Produtos:** Visualização rápida de itens disponíveis.
- **Contato via WhatsApp:** Link direto para negociação com o produtor.
- **Cadastro Simples:** Formulário intuitivo para anúncio de novos produtos.

## 🛠️ Tecnologias Utilizadas
* [Python](https://www.python.org/)
* [Flask](https://flask.palletsprojects.com/) (Micro-framework web)
* [SQLite](https://www.sqlite.org/) (Banco de dados leve)
* [Jinja2](https://jinja.palletsprojects.com/) (Templates HTML)
